Ebitengine Game Jam started

Hajime Hoshi
2022-06-15

The Ebitengine game jam has started today! The theme for this event is a magnet 🧲!

A game jam is an event to develop a game in a defined time period (2 weeks for this event), show off yours, and evaluate each other's. Let's join this event and have fun! For further questions, please ask at the Ebitengine Discord.

I appreciate Siôn le Roux for managing the jam!
