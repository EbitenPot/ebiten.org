v2.3.0 is released

Hajime Hoshi
2022-04-26

I am very happy to announce the release of v2.3.0 (Release Note).

I appreciate all the contributors and all the sponsors. Thank you very much!

v2.3.0 includes these new features:

  • Supporting DirectX on Windows
  • Native compile for Nintendo Switch™
  • A build tag ebitenexternaldll
  • Device vibration
  • Real-time PCM
  • etc.

v2.3.0 also improves performance in terms of speed and memory usage.

I appreciate your feedbacks!

v2.3.0 がリリースされました

Hajime Hoshi
2022-04-26

v2.3.0 がリリースされました! 詳しくはリリースノートを参照してください。

すべてのコントリビューターとスポンサーの皆様に感謝いたします。どうもありがとうございます!

v2.3.0 に入った新しい機能は次のとおりです:

  • Windows の DirectX サポート
  • Nintendo Switch™ 向けネイティブコンパイル
  • ビルドタグ ebitenexternaldll
  • デバイスのバイブレーション
  • リアルタイム PCM
  • その他

v2.3.0 で、スピードやメモリ使用量などのパフォーマンス改善も行いました。
