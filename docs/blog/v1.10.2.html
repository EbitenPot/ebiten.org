v1.10.2 is released

Hajime Hoshi
2019-12-26

We are very happy to announce the release of v1.10.2. This includes only bug fixes from v1.10.1.

  • Compile error with Go 1.14 (beta) for Wasm (Issue #1024)
  • Avoid unnecessary copying: this is a performance improvement rather than a bug fix, but this was needed to pass on the test bots. (Issue #983)
