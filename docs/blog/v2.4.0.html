v2.4.0 is released

Hajime Hoshi
2022-08-31

I am very happy to announce the release of v2.4.0 (Release Note).

I appreciate all the contributors and all the sponsors. Thank you very much!

v2.4.0 includes these new features:

  • "Truly pure Go" for Windows
  • Unmanaged images
  • Audio decoding without resampling
  • Debug info
  • etc.

v2.4.0 also improves performance in terms of speed and memory usage, especially in the DirectX driver.

I appreciate your feedbacks!

v2.4.0 がリリースされました

Hajime Hoshi
2022-08-31

v2.4.0 がリリースされました! 詳しくはリリースノートを参照してください。

すべてのコントリビューターとスポンサーの皆様に感謝いたします。どうもありがとうございます!

v2.4.0 に入った新しい機能は次のとおりです:

  • Windows での「真の Pure Go」
  • アンマネージドな画像
  • リサンプリングなしのオーディオデコード
  • デバッグ情報
  • その他

v2.4.0 で、特に DirectX ドライバにおいて、スピードやメモリ使用量などのパフォーマンス改善も行いました。

フィードバックお待ちしております!
