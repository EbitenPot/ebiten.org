v1.10.1 is released

Hajime Hoshi
2019-11-24

We are very happy to announce the release of v1.10.1. This includes only bug fixes from v1.10.0.

  • RepalcePixels might be very slow unexpectedly (Issue #977).
  • CPU was busy when the window is in another workspace (Issue #982).
  • Delayed sound on macOS / iOS (Issue #975).
  • etc.
